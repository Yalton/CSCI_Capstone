# Computer science capstone project

## About 
- Software designed to scan potholes and calculate volume 
- Using the calculated volume and density provided by user mass of required material can be calculated

## Python Requirements
- Version=3.9
- tkinter
- numpy
- matplotlib
- open3d
- intelrealsenseSDK
